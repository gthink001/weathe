Wi-Fi Enable
1. Over The Internet - you can control any appliances from anywhere anytime when connected to internet
2. Locally When connected to Wi-Fi Range - Without internet you can connect the device to the Wi-Fi signal and can control all the appliances with in the Wi-Fi range
3. Individually -Even without Wi-Fi router you can individually connect to our device hotspot signals and can control the appliance connected.
Retrofit: Existing switches can be used easily without any changes to the electric wiring.
Physical(Manual)Control - Appliances can be controlled using wall-mounted traditional switches.
G-Smart App Control: G-Smart Application can be downloaded on Android and IOS can be operated from anywhere in the world.
Voice enable control: G-Smart Switches are compatible with Voice-enabled devices like Amazon Alexa.
