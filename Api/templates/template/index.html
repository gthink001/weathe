WHY CUSTOMERS CHOOSE US

Our device is designed for the manual switches to work independently when connected to the circuit, which works as a backup when an issue takes place in any rare case.
We designed the Fan Regulator which rotates automatically when operated from mobile and gives a full feedback even when controlled manually. We can jump from any speed to any speed from the app as well as through manual switch.
Individual switch can be selected to connect it with the appliances when the device is connected through inverter.
We provide live status of the appliances connected to the device, by which we generate monthly report to the customers which shows the exact load consumption of the appliances.
Our device has inbuilt serge protection circuit, which protects itself from any high voltages. By which all the appliances connected to it also protected.
Our same device can be controlled via internet and also through intranet, it has a feature to connect to our individual device hotspot signal and controlled if there is no router available.
