APP FEATURES









HOME
In the home page, we can see the list of rooms, of all the devices which are installed in a home.





Room
The room consists of different applications, with that we can able to see and control the appliances from the G-Smart Home Application.



Timer
This feature is used to set the timer to any home appliances which can be ON or OFF at a specified time within the day.








Schedule
This feature is used to schedule all your connected appliances to go ON and OFF automatically for your everyday tasks and relax while the app works for you even at your absense. Scheduling task can also take place independently without the need of internet, in offline.



Household
This feature is used to provide complete control of all the rooms to specific registered members in a family. Further, unlimited members can be added as an individual household .






Switch Mode
This feature is used to create an own mode and control multiple appliances with a single tap by assigning the action of multiple switches to a single switch.











Turn ON All
With this feature, we can Turn ON all the appliances with a single click. Turn OFF All: With this feature, we can Turn OFF all the appliances with a single click.
